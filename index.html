Viatran

View Site

Viatran is a manufacturer of pressure transmitters based in the United States. We took a legacy, non-responsive website that was had an outdated CMS and build a modern, custom CMS around it and ported in the existing database and content.

  • Provided a CMS that allows edits to edit most content on the site. The CMS also features a media management system.
  • Performed content migration, frontend execution, and backend execution.
  • Powered by Django+React.

Keysight

View Site

Keysight Technologies is a large electronic measurement devices company based in the United States. We took a legacy, non-responsive website that was not backed by any CMS and build a custom CMS around it and ported in the existing database.

  • Provided a CMS that allows edits to edit nearly all content on the site. The CMS also features a media management and publishing system.
  • Performed content migration, frontend execution, and backend execution.
  • Powered by Django+React.
